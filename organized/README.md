# BOSS SHOPP - Organized Structure

This is the organized version of the BOSS SHOPP project with a clear folder structure.

## Folder Structure

```
organizado/
├── backend/          # Django backend application
├── frontend/         # Frontend files (HTML, CSS, JavaScript)
├── database/         # Database files and schemas
├── docs/             # Documentation
│   ├── setup/        # Setup instructions
│   ├── api/          # API documentation
│   └── user_guide/   # User guides
├── scripts/          # Utility scripts
└── tests/            # Test files
```

## Description

- **backend/**: Contains all Django backend code, API implementation, and server-side logic
- **frontend/**: Contains all HTML, CSS, JavaScript, and static assets for the frontend
- **database/**: Contains database schemas, migration files, and database management scripts
- **docs/**: Contains all project documentation
- **scripts/**: Contains utility scripts for setup, deployment, and maintenance
- **tests/**: Contains test files for both frontend and backend

This organization makes it easier to navigate and maintain the project.

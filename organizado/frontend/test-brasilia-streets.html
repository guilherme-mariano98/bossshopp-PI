Ruas de Brasília

API de CEP - Todas as ruas do Distrito Federal

Sobre esta API

Esta API fornece uma lista completa de ruas e setores da cidade de Brasília (Distrito Federal). É útil para preenchimento automático de endereços e validação de dados.

Endpoint da API

GET http://localhost:5001/api/brasilia/streets

Observação: Certifique-se de que o serviço de CEP esteja em execução na porta 5001.
